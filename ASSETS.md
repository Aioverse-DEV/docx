# Assets Directory

This directory contains static assets for the documentation.

## Structure

- `logo/` - Logo files (light.svg and dark.svg)
- `images/` - General images used in documentation
- `public/` - Public assets accessible in documentation

## Logo Files

You should add the following logo files:
- `logo/light.svg` - Logo for light mode
- `logo/dark.svg` - Logo for dark mode
- `favicon.svg` - Favicon

Alternatively, you can use other image formats (PNG, JPG) and update the paths in `mint.json`.

## Images

Place any images referenced in your MDX files here. Reference them in your documentation like:

```mdx
![Alt text](/images/your-image.png)
```

## Hero Images

The introduction page references hero images. Add them here:
- `images/hero-light.svg` - Hero image for light mode
- `images/hero-dark.svg` - Hero image for dark mode
