## Scope

This repository provides a single source of truth for:

- Multi-source ingestion: laptop, phone, links, pdf, docs, md
- Infrastructure documentation: Cloudflare origin, DNS records, server configs
- GitHub and enterprise structure documentation
- Branding, templates, and reusable code snippets
- AI agent conversation auto-documentation workflows
