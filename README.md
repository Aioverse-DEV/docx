## 🎨 AIOVERSE Brand Theme

This repository includes the official **AIOVERSE brand theme** configuration for use across all Aioverse-DEV repositories, tasks, and jobs.

### Brand Colors

| Color | Hex | Usage |
|-------|-----|-------|
| **Electric Aqua** | `#73effb` | Primary accent, CTAs, highlights |
| **Midnight Teal** | `#002428` | Primary backgrounds, dark mode |
| **Soft White** | `#fafafa` | Text on dark, light backgrounds |

### Quick Start

```html
<!-- Use in HTML -->
<link rel="stylesheet" href="theme/colors.css">
<div class="aio-theme-dark">
  <h1 class="aio-text-primary">AIOVERSE</h1>
</div>
```

```javascript
// Use in JavaScript
import { AIOVERSEColors } from './theme/theme.js';
const primaryColor = AIOVERSEColors.ELECTRIC_AQUA; // #73effb
```

```python
# Use in Python
from theme.theme import AIOVERSEColors
AIOVERSEColors.print_branded("Success!", "electric_aqua")
```

See [theme/USAGE.md](theme/USAGE.md) for complete usage documentation.
