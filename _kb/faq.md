# Frequently Asked Questions (FAQ)

## General Questions

### What is Aioverse?

Aioverse is a comprehensive development platform that provides tools, libraries, and documentation to help developers build better applications faster.

### Is Aioverse free to use?

Yes, Aioverse is open source and free to use under the terms specified in the LICENSE file.

### What programming languages are supported?

Aioverse supports multiple programming languages including:
- JavaScript/TypeScript
- Python
- Go
- Rust

## Technical Questions

### How do I install Aioverse?

See our [Getting Started Guide](/docx/blogs/2026-04-16-getting-started/) for detailed installation instructions.

### Can I contribute to Aioverse?

Absolutely! We welcome contributions. Please read our contributing guidelines in the repository.

### Where can I report bugs?

Report bugs by opening an issue on our [GitHub repository](https://github.com/Aioverse-DEV/docx/issues).

### How do I get support?

You can get support through:
- This documentation site
- GitHub Discussions
- Community forum
- Stack Overflow (tag: aioverse)

## Licensing

### What license does Aioverse use?

Aioverse is licensed under the Apache License 2.0. See the LICENSE file for full details.

### Can I use Aioverse in commercial projects?

Yes, the Apache License 2.0 allows commercial use.
