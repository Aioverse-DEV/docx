Welcome to the Aioverse Docx blog! This is your personal space for:

- **Knowledge Base**: Store and organize your technical knowledge
- **Notes**: Quick thoughts and ideas
- **Wiki**: Build your personal wiki
- **Library**: Collection of useful resources

## Getting Started

This blog is built with Jekyll and hosted on GitHub Pages. You can:

1. Write posts in Markdown format in the `_posts` directory
2. Customize the design by editing layouts and CSS
3. Add pages for different sections (KB, Notes, Wiki, Library)

## Writing Posts

Create new posts with the naming convention: `YYYY-MM-DD-title.md`

Example:
```
2026-04-16-my-first-post.md
```

Happy blogging!
