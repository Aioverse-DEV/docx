# AI Agent Conversation Auto-Documentation

## Objective

Automatically capture AI agent conversations as durable documentation artifacts.

## Workflow

1. Capture conversation transcript ID/source.
2. Extract decisions, actions, risks, and follow-ups.
3. Generate summary document in markdown.
4. Store in `/wiki/ai-conversations/` or `/docs/ai/records/`.
5. Link summary to related runbook/issue/PR.
6. Add reviewer sign-off for high-impact changes.

## Minimum summary schema

- Context
- Problem statement
- Options considered
- Chosen decision
- Implementation notes
- Security/compliance notes
- Open items
