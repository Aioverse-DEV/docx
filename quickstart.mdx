## Setup your environment

Learn how to set up your documentation environment and start creating content.

<AccordionGroup>
  <Accordion icon="github" title="Clone the repository">
    First, clone the repository to your local machine:
    ```bash
    git clone https://github.com/Aioverse-DEV/docx.git
    cd docx
    ```
  </Accordion>

  <Accordion icon="download" title="Install dependencies">
    Install the required dependencies:
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```
  </Accordion>

  <Accordion icon="play" title="Run locally">
    Start the development server:
    ```bash
    mintlify dev
    ```
    Your documentation will be available at `http://localhost:3000`
  </Accordion>
</AccordionGroup>

## Add content

<Steps>
  <Step title="Create a new page">
    Create a new `.mdx` file in your documentation folder:
    ```bash
    touch my-new-page.mdx
    ```
  </Step>
  <Step title="Add frontmatter">
    Add the required frontmatter at the top of your file:
    ```mdx
    ---
    title: 'My New Page'
    description: 'Description of my new page'
    ---
    ```
  </Step>
  <Step title="Update navigation">
    Add your new page to the navigation in `mint.json`:
    ```json
    "navigation": [
      {
        "group": "Get Started",
        "pages": ["introduction", "my-new-page"]
      }
    ]
    ```
  </Step>
  <Step title="Deploy">
    Push your changes to deploy:
    ```bash
    git add .
    git commit -m "Add new page"
    git push
    ```
  </Step>
</Steps>
