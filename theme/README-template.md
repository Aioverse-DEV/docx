## 🎨 Brand Guidelines

This project follows the **AIOVERSE brand guidelines**:

- **Colors**: Electric Aqua (#73effb), Midnight Teal (#002428), Soft White (#fafafa)
- **Typography**: NEBULA 2 (display), Nohemi (headings), Helvetica Neue (body)
- **Design Principles**: Dark-field first, signal over noise, geometric tension

See the [Aioverse Branding Repository](https://github.com/Aioverse-DEV/Aioverse-Branding) for full guidelines.
